Vínculo com o GitHub

Passo a passo de como fazer o vínculo d VScode com o Git

Iniciando no terminal com os seguintes comandos:
  1. sudo chmod -R 777 code (para desbloquear a pasta Code)
  2. sudo rm -r code (para excluir a pasta code)
  3. Dar o mesmo comando para excluir a pasta 'design'
  4. No VScode abrir a pasta da loja e dentro dela, abrir a src/app
  5. Dar o comando no terminal 'git init' dentro da pasta app da loja
  6. Ainda no VScode:
    1. Acessar a 3ª view
    2. Clicar nos 3 pontinhos (...)
    3. Remote
    4. Adicionar
    5. Copiar a URL do repositório da loja no Git
    6. Colar essa URL na aba que foi aberta no VScode
    7. Nomear
  7. Ainda na 3ª view, desfazer as mudanças
  8. Acessar a branch indicada e cria sua própria branch para fazer as alterações
  9. No terminal, voltar paa a pasta raiz da loja
  10. Dar o comando 'bin/restart' para iniciar a loja
  11. Por fim, dar o comando bin/magento c:c (para limpar o cash)

Adicionando o tema ao site

  1. O primeiro passo é acessar o link do administrador do Painel Magento
  2. View 'Content'
  3. Configurações
  4. Selecionar a 3ª opção da tabela e clicar em 'Edit'
  5. Alterar o tema
  6. E após isso, limpar mais uma vez o cash pelo terminal

Caso ocorer algum erro ou a página não carregar, tente os seguintes comandos no terminal

  • bin/magento mo:s (para verificar se tem módulo desabilitados além do TwoFactorAuth)
  • bin/magento module:enable --all (se tiver mais módulos desabilitados, esse comando irá habilitar todos)
  • bin/magento module:disable Magento_TwoFactorAuth (para desabiliá-lo novamente, pois este é o único que deve estar desabilitado)

Após comandos desse tipo, é necessário dar um upgrade e compile, conforme os comandos abaixo

  • bin/magento s:up
  • bin/magento s:d:c
  • bin/magento c:c

E se após esses comandos ainda ocorrer erro, dê o comando abaixo:

bin/composer require mageplaza/module-core
